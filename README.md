# *SmartHomeAutomation*
</br>
The purpose of this project is to simulate a smart home running on Arduino.
</br>
The project was carried out on the Proteus program. (new project file)
</br>
Within the system;
• Fire alarm,
• Motion detecting light system,
• Digital thermometer,
• There is a lock system.
